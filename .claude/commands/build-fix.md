# /build-fix

ビルドエラーを修正するコマンド。

## 実行手順

1. **ビルド実行**
```bash
bun run build
# または
bun run check-types
```

2. **エラー解析**
   - ファイル別に分類
   - 重要度で優先順位付け

3. **修正サイクル**
   - エラーを1つ選択
   - コンテキストを表示
   - 問題を説明
   - 修正を提案・適用
   - ビルド再実行で検証

4. **完了まで繰り返し**

## 原則

- **1エラーずつ**: カスケードエラーを防ぐ
- **最小限の変更**: リファクタリングしない
- **検証必須**: 各修正後にビルド確認

## 停止条件

- 修正が新しいエラーを生む
- 同じエラーが3回再発
- ユーザーが停止を指示

## 完了報告

```markdown
## ビルド修正完了

### 修正済み
- [ファイル:行] - 問題と解決策

### 未解決
- [ファイル:行] - 理由

### 新規発生
- なし（あれば記載）
```
